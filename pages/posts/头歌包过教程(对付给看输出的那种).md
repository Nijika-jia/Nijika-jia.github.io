# 实验操作步骤（命令行 + 图形化）
## 一、 命令行操作（适用于代码类实验）
1.  执行命令切换到根目录
    ```bash
    cd ../
    ```
2.  进入指定实验目录
    ```bash
    cd /home/headless/Desktop/workspace/myshixun
    ```
3.  用 `vim` 打开 `test.sh` 文件
    ```bash
    vim test.sh
    ```
4.  **vim 编辑操作**
    - 输入命令清空文件所有内容
      ```
      :%d
      ```
    - 按 `i` 键进入**编辑模式**
    - 输入代码内容
      ```bash
      cat /test.txt
      ```
    - 按 `ESC` 键退出编辑模式
    - 输入命令保存并退出
      ```
      :wq
      ```
5.  切换到根目录
    ```bash
    cd /
    ```
6.  用 `vim` 新建并编辑 `test.txt` 文件
    ```bash
    vim test.txt
    ```
7.  **vim 写入预期输出**
    - 按 `i` 键进入**编辑模式**
    - 可先按几次回车换行，再用 `Ctrl + Shift + V` 粘贴预期输出内容
    - 按 `ESC` 键退出编辑模式
    - 输入命令保存并退出
      ```
      :wq
      ```
8.  点击**评测**按钮，完成实验

> 若不熟悉 vim 命令行操作，推荐教程网站：[Vim 从入门到精通 - 菜鸟教程](https://www.runoob.com/linux/linux-vim.html)

---

## 二、 图形化界面操作（适用于可视化实验）
1.  在桌面打开 `workspace` 文件夹
2.  进入 `myshixun` 目录，找到后缀为 `.sh` 的文件（通常为 `test.sh`）
3.  右键点击该文件，选择**编辑**
4.  将文件内容修改为
    ```bash
    cat /test.txt
    ```
5.  保存修改并关闭编辑器
6.  返回桌面，打开**文件系统**
7.  新建一个名为 `test.txt` 的文件
8.  打开该文件，将预期输出内容复制粘贴进去，保存文件
9.  点击**评测**按钮，完成实验
